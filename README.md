# Universal Bank — Streamlit Cloud App

**Files (no folders):**
- `app.py` — main Streamlit application
- `requirements.txt` — minimal libs (no pinned versions)

## Run locally
pip install -r requirements.txt
streamlit run app.py

## Tabs
- Insights (5 charts)
- Modeling (3 algorithms + metrics, ROC, confusion, importances)
- Predict (upload & download predictions)
